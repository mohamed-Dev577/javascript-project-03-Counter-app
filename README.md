# 🔢 Counter App

مشروع بسيط باستخدام JavaScript لإنشاء عداد أرقام يمكن زيادته أو تقليله أو إعادة ضبطه.

هذا المشروع يساعدك على فهم أساسيات البرمجة مثل المتغيرات والتفاعل مع المستخدم.

---

## 🚀 فكرة المشروع

الصفحة تحتوي على:

- رقم يظهر على الشاشة
- زر لزيادة الرقم (+)
- زر لتقليل الرقم (-)
- زر لإعادة الرقم إلى الصفر (Reset)

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript

---

## 🎯 ماذا تعلمت من المشروع؟

- إنشاء متغير باستخدام `let`
- استخدام العمليات الحسابية (`++` و `--`)
- التعامل مع الأحداث `onclick`
- تحديث المحتوى داخل الصفحة باستخدام `textContent`
- ربط JavaScript بعناصر HTML
